ES6 Tasks

Task 1: Convert Array to Arrow Functions

Convert the following functions to arrow functions:

function square(x) {
  return x * x;
}

function greet(name) {
  return `Hello, ${name}!`;
}

Task 2: Destructuring Assignment

Given the following array, use destructuring assignment to extract the values into separate variables:

const colors = ['red', 'green', 'blue'];

Task 3: Spread Operator

Use the spread operator to combine the two arrays into a single array:

const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];
